# 📱 모바일 링크 명함

React + Vite + MUI로 만든 모바일 최적화 링크 명함입니다.

## ✨ 주요 기능

- 📱 **모바일 최적화**: 모바일과 태블릿에 최적화된 반응형 디자인
- 🎨 **MUI 컴포넌트**: Material-UI를 활용한 현대적인 디자인
- 📂 **자동 포트폴리오**: `src/portfolio` 폴더에 이미지만 넣으면 자동으로 표시
- 🔄 **토글 기능**: 자기소개와 경력 섹션을 버튼으로 접기/펼치기
- 📱 **터치 최적화**: 모바일 터치에 최적화된 UI

## 🚀 시작하기

### 1. 의존성 설치
```bash
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
```

### 3. 포트폴리오 이미지 추가
`src/portfolio` 폴더에 이미지 파일들을 넣으면 자동으로 포트폴리오 섹션에 표시됩니다.

## 📁 프로젝트 구조

```
mobile_profile/
├── src/
│   ├── portfolio/     # 포트폴리오 이미지들 (자동 로드)
│   ├── App.jsx        # 메인 컴포넌트
│   ├── App.css        # 스타일
│   └── main.jsx       # 앱 진입점
├── public/            # 정적 파일
└── package.json       # 프로젝트 설정
```

## 🎯 사용법

1. **프로필 정보 수정**: `src/App.jsx`의 `profile` 객체를 수정
2. **포트폴리오 이미지**: `src/portfolio/` 폴더에 이미지 파일 추가
3. **스타일 커스터마이징**: `src/App.css`에서 스타일 수정

## 📱 반응형 지원

- **모바일**: 320px ~ 768px
- **태블릿**: 768px ~ 1024px
- **데스크톱**: 1024px 이상

## 🛠️ 기술 스택

- **React 18** - UI 라이브러리
- **Vite** - 빌드 도구
- **Material-UI (MUI)** - UI 컴포넌트 라이브러리
- **CSS3** - 스타일링
